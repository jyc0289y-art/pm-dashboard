# Cf-252 선량평가 및 방사화평가 계산기 개발 지침서

> **목적**: 호진산업기연(주)의 Cf-252 방사선안전보고서(REV13, 6차 보완) 최종 평가방식을 기반으로,  
> 회의 시 즉각 대응 가능한 선량평가 및 방사화평가 계산기를 개발하기 위한 기술 사양서

---

## 1. 개요

### 1.1 평가 대상 핵종
- **Cf-252** (Californium-252)
- 주요 특성: 자발핵분열(Spontaneous Fission) 선원 → 중성자 + 감마선 **혼합방사선장** 형성

### 1.2 평가 시나리오 (3가지)
1. **저장시설**: 운반용기에 내장된 Cf-252를 저장구역에 보관 시 시설 표면 선량률
2. **핫셀 저장/취급**: 핫셀 내부에서 Cf-252를 저장함에 보관 또는 매니퓰레이터로 취급 시 선량률
3. **방사화 평가**: 핫셀 매니퓰레이터(Al-27) 집게의 중성자 방사화

### 1.3 평가 방법론 핵심
- **MCNP 없이 해석적(analytical) 방법** 사용
- 감마선과 중성자를 **분리하여** 각각 평가 후 합산
- 점선원 근사(point source approximation) + 역제곱법칙 + HVL 감쇠

## 3. 감마선 선량률 평가 (최종 방법)

### 3.1 감마상수 (Exposure Rate Constant)
- **출처**: Smith, D. S., & Stabin, M. G. (2012). *Exposure rate constants and lead shielding values for over 1,100 radionuclides.* Health Physics, 102(3), 271–291.
- **ICRP-107 기반 핵데이터** 사용 (ICRP-38 라이브러리는 160keV까지만 반영하여 부적합 → **6차 보완에서 변경**)

```
Cf-252 감마상수 (Γ_γ):
  SI 단위:    4.48 × 10⁻¹³ C·m² / (kg·MBq·s)
  전통 단위:  2.31 R·cm² / (mCi·h)
  f-factor:   0.960 cGy/R
```

#### 감마상수의 단위 변환 (계산기 구현용)
보고서에서 사용하는 실무 단위: `mSv·cm² / (mCi·h)`

변환:
```
2.31 R·cm²/(mCi·h) × 0.960 cGy/R × (1 mSv / 0.1 cGy) = 22.176 mSv·cm²/(mCi·h)
```

> **주의**: 보고서 본문에서 구체적인 감마상수 수치 표기는 첨부문서 p.56에 기재되어 있으며,  
> Smith & Stabin 논문의 Cf-252 행에서 직접 인용한 값이다.
> 보고서 첨부1 p.56에 기재된 감마상수 표기: `Γ = [수치] × 10⁻¹ mSv·cm²/(mCi·s)` 형태

### 3.2 감마선 선량률 기본 공식
```
Ḋ_γ = Γ_γ × A / d²
```
- `Γ_γ`: Cf-252 감마상수
- `A`: 방사능량 (mCi)
- `d`: 평가 거리 (cm)

### 3.3 차폐 감쇠 (HVL 방식)
```
Ḋ_γ(차폐 후) = Ḋ_γ(무차폐) × (1/2)^(t/HVL)
```
- `t`: 차폐체 두께 (cm)
- `HVL`: 반가층 (cm)

**다층 차폐 시:**
```
감쇠인자 = (1/2)^(t₁/HVL₁) × (1/2)^(t₂/HVL₂) × ...
```

## 4. 중성자 선량률 평가 (최종 방법)

### 4.1 중성자 선속밀도 당 선량률 환산계수
- **출처**: 정덕연, 장시영, 윤석철, & 김종수. (1993). 비감속 Cf-252 중성자선원에 대한 비등방성교정인자 및 선량당량환산인자. *방사선 방어학회지*, 18(2), 71–78.

```
H/Φ = 332.9 pSv·cm²/neutron
```

> ISO 8529 권고값 340 pSv·cm²/neutron과 약 1.8% 차이로 검증됨

### 4.2 Cf-252 1 mCi의 중성자 선속밀도 유도

#### Step 1: 단위시간 당 붕괴 횟수
```
λ = 3.7 × 10⁷ dis/s  (1 mCi 기준)
```

#### Step 2: 단위시간 당 중성자 방출 수
```
N_n = λ × BF_SF × ν
    = 3.7 × 10⁷ × 0.03092 × 3.765
    = 4.309 × 10⁶ neutrons/s  (per mCi)
```

#### Step 3: 1 cm 거리에서 중성자 선속밀도
```
Φ(1cm) = N_n / (4π × r²)
        = 4.309 × 10⁶ / (4π × 1²)
        = 3.429 × 10⁵ neutrons/(cm²·s)  (per mCi)
```

#### Step 4: 중성자 선량률 환산계수 유도
```
Ḣ_n(1cm) = Φ(1cm) × (H/Φ)
          = 3.429 × 10⁵ × 332.9 × 10⁻¹²
          = 1.141 × 10⁻⁴ mSv/s  (per mCi at 1cm)
```

**실무 단위로 정리:**
```
Cf-252의 중성자 선량률 환산계수 = [계산 결과] mSv·cm²/(mCi·h)
```

> 보고서 첨부1 p.58에서 이 유도 과정을 상세히 기술하고 있으며,  
> 최종적으로 `mSv·cm²/(mCi·h)` 단위의 환산계수를 도출한다.

### 4.3 중성자 선량률 기본 공식
```
Ḣ_n = Γ_n × A / d²
```
- `Γ_n`: 위에서 유도된 중성자 선량률 환산계수
- `A`: 방사능량 (mCi)
- `d`: 평가 거리 (cm)

### 4.4 차폐 감쇠 (HVL 방식)
```
Ḣ_n(차폐 후) = Ḣ_n(무차폐) × (1/2)^(t/HVL)
```

## 9. 방사화 평가 (Al-27 → Al-28)

### 9.1 개요
핫셀 매니퓰레이터 집게(Al-27 재질)가 Cf-252 중성자에 의해 방사화되는 정도를 평가

### 9.2 핵반응
```
Al-27 (n, γ) Al-28
```

### 9.3 Al-28 핵데이터
```
생성핵종:     Al-28
반감기:       2.2414 분 (약 134.5 초)
붕괴모드:     β⁻ 붕괴 → Si-28 + β⁻ + γ
주요 감마선:  1.779 MeV (100%)
```

### 9.4 반응단면적
```
Al-27의 (n,γ) 반응단면적 @ 2 MeV 대역:
  σ = [ENDF/B 데이터에서 인용한 값] (cm²)
```
> 보고서 p.97에서 Cf-252 평균 에너지대역인 2MeV에서의 Al-27 (n,γ) 단면적을 사용  
> 실제 값은 약 0.5 ~ 1 mb 수준 (빠른 중성자 대역)

### 9.5 방사화 공식

```
A_activated = N_target × σ × Φ × (1 - e^(-λ × t))
```

여기서:
- `N_target`: 조사 영역 내 Al-27 원자 수 (보고서에서는 선원에 직접 노출되는 집게 부분)
- `σ`: (n,γ) 반응단면적 (cm²)
- `Φ`: 중성자 선속밀도 (neutrons/cm²·s)
- `λ`: Al-28의 붕괴상수 = ln(2) / T½ = 0.693 / 134.5 s
- `t`: 조사시간

### 9.6 보고서의 방사화 계산 공식 (원문 방식)

```
A = λ × N × σ_a × φ × (1 - e^(-λt))
```

보고서에서는 다음과 같이 전개:

```
A = (BF_SF × ν × 붕괴율) × σ × (1 - e^(-λt)) / (4π × r²)
```

여기서 중성자 선속밀도(φ)는 선원 방사능량으로부터 직접 계산:
```
φ = A_source × 3.7×10⁷ × BF_SF × ν / (4π × r²)
```

### 9.7 조사 가정
```
조사시간: 1분 (선원 1개 옮기는 시간)
조사거리: 선원에서 집게까지 거리 (밀착 취급)
냉각시간: 0 (무시 → 보수적 가정)
차폐:     없음 (무차폐 → 보수적 가정)
```

### 9.8 Al-28 감마상수
```
출처: MicroShield10 + ICRP107 데이터
Al-28 감마상수 (Γ): [보고서 p.97에 기재된 값] × 10⁻¹ mSv·cm²/(mCi·s)
```

> Smith & Stabin 논문에서 Al-28:
> - Exposure rate constant = 1.62 × 10⁻¹² C·m²/(kg·MBq·s) = 8.37 R·cm²/(mCi·h)
> - f-factor = 0.876 cGy/R

### 9.9 생성핵종 선량률
```
Ḋ_Al28 = Γ_Al28 × A_activated / d²
```
- 평가 거리: 1m (무차폐)
- 판정기준: 1 μSv/h 미만

## 13. 주의사항 및 한계

1. **HVL 방법의 한계**: 단일 에너지 감쇠에 기반한 근사. Cf-252의 연속 스펙트럼에서 빌드업 효과가 과소평가될 수 있으나, 충분한 차폐 두께(>20 HVL)에서는 보수적 결과.

2. **점선원 근사**: 선원 크기 대비 평가 거리가 충분히 클 때 유효. 밀착 취급 시에는 오차 증가 가능.

3. **방사화 평가의 근사**: 2 MeV 대역의 대표 단면적 사용은 spectrum-averaged cross section 대비 근사적이나, 열중성자화가 없는 환경에서 실무적으로 수용 가능.

4. **보수적 가정의 방향**: 냉각시간 무시, 무차폐 조건 등은 모두 실제보다 높은 선량을 산출하는 보수적 방향.

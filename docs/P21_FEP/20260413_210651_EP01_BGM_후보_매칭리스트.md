## BGM-A' | 메인 테마 풍성 변주 (별도 소싱 불요)

- 그렇지 않으면 BGM-A에 리버브 + 패드 레이어를 FFmpeg로 추가하여 A' 생성

## BGM-B' | 일상 루프 불안 변형 (별도 소싱 불요)

**용도**: S-06 후반 — 같은 멜로디, 반음 하강/리듬 어긋남

BGM-B로 선정된 트랙을 FFmpeg로 가공:
```bash
# 반음 내림 (pitch shift -1 semitone)
ffmpeg -i bgm_b.mp3 -af "asetrate=44100*0.9439,aresample=44100" bgm_b_prime.mp3
```

## 다음 단계

1. **시청 검수**: 각 1순위 후보를 Pixabay에서 미리듣기하여 실제 분위기 확인
2. **길이/구간 확인**: 필요 구간(초)과 트랙 길이 대조 — 루프/트림 필요 여부 판단
3. **다운로드**: 확정 트랙 MP3 다운로드 → `EP01_forced_compliment/audio/bgm/` 저장
4. **BGM-B'/F' 가공**: FFmpeg pitch shift + 음 제거로 변형 트랙 생성
5. **SFX 소싱**: freesound.org + pixabay.com/sound-effects 에서 SFX-1~8 매칭 (별도 작업)

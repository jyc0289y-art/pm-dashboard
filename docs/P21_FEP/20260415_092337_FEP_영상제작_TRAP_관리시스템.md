## 1. 일러스트 생성

### TRAP-001: txt2img 단독 캐릭터 생성
- **발견 일시**: 2026-04-14
- **발견 Phase**: Phase 8d
- **원인 파일**: ComfyUI txt2img 워크플로우
- **수정 파일**: `generate_scenes_v2.py` (CHAR_* 상수 추가)
- **증상**: txt2img로 캐릭터를 생성하면 매번 얼굴, 체형, 의상이 달라짐
- **원인**: txt2img는 프롬프트만으로 이미지를 생성하므로 캐릭터 일관성 유지 불가
- **해결**: **캐릭터 시트 img2img 참조 필수** (denoise 0.3~0.5). Style B는 캐릭터 시트를 프롬프트에 상세 기술
- **예방**: 캐릭터 시트 없이 씬 생성 절대 금지. 에피소드 시작 시 캐릭터 시트 먼저 확보
- **상태**: ✅ 해결됨

---

### TRAP-002: 완성 씬 일괄 생성
- **발견 일시**: 2026-04-14
- **발견 Phase**: Phase 8d
- **원인 파일**: 초기 생성 파이프라인
- **수정 파일**: `v3_composite_portrait.py`, `v3_composite_scenes.py`
- **증상**: 배경+캐릭터를 한 이미지로 생성 → 위치/크기 제어 불가, 배경 불일치, LP 적용 불가, 재사용 불가
- **원인**: 단일 이미지 내에서 개별 요소 제어가 구조적으로 불가능
- **해결**: **레이어 분리 생성** — 배경/캐릭터/소품을 독립 생성 후 Pillow 합성
- **예방**: EP.02부터 완성 씬 일괄 생성 방식 완전 폐기
- **상태**: ✅ 해결됨

### TRAP-012: concat glob 임시 파일 포함 → 영상 부풀림
- **발견 일시**: 2026-04-14
- **발견 Phase**: Phase 8f

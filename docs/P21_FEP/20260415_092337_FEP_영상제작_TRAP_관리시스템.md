## 1. 일러스트 생성

### TRAP-001: txt2img 단독 캐릭터 생성
- **발견 일시**: 2026-04-14
- **발견 Phase**: Phase 8d
- **원인 파일**: ComfyUI txt2img 워크플로우
- **수정 파일**: `generate_scenes_v2.py` (CHAR_* 상수 추가)
- **증상**: txt2img로 캐릭터를 생성하면 매번 얼굴, 체형, 의상이 달라짐
- **원인**: txt2img는 프롬프트만으로 이미지를 생성하므로 캐릭터 일관성 유지 불가
- **해결**: **캐릭터 시트 img2img 참조 필수** (denoise 0.3~0.5). Style B는 캐릭터 시트를 프롬프트에 상세 기술
- **예방**: 캐릭터 시트 없이 씬 생성 절대 금지. 에피소드 시작 시 캐릭터 시트 먼저 확보
- **상태**: ✅ 해결됨

---

### TRAP-002: 완성 씬 일괄 생성
- **발견 일시**: 2026-04-14
- **발견 Phase**: Phase 8d
- **원인 파일**: 초기 생성 파이프라인
- **수정 파일**: `v3_composite_portrait.py`, `v3_composite_scenes.py`
- **증상**: 배경+캐릭터를 한 이미지로 생성 → 위치/크기 제어 불가, 배경 불일치, LP 적용 불가, 재사용 불가
- **원인**: 단일 이미지 내에서 개별 요소 제어가 구조적으로 불가능
- **해결**: **레이어 분리 생성** — 배경/캐릭터/소품을 독립 생성 후 Pillow 합성
- **예방**: EP.02부터 완성 씬 일괄 생성 방식 완전 폐기
- **상태**: ✅ 해결됨

### TRAP-012: concat glob 임시 파일 포함 → 영상 부풀림
- **발견 일시**: 2026-04-14
- **발견 Phase**: Phase 8f
- **원인 파일**: `build_video.py` concat 로직
- **수정 파일**: `build_video.py` (glob 필터 강화)
- **증상**: `_seg`, `_cut`, `_bcut` 임시 파일이 glob에 포함 → 이중 concat → 133초 부풀림

- **예방**: concat 대상 파일 목록을 빌드 전 반드시 출력하여 검증
- **상태**: ✅ 해결됨

  - "나 예쁘지?", "응, 예뻐", "오늘 괜찮지?", "응, 괜찮아" 모두 같은 세그먼트 시작(0초)으로 snap
- **원인**: silencedetect는 **"발화의 시작/끝"만 감지하고 화자/내용을 구분 못함**. "가장 가까운 발화"는 단순 시간 거리 기준이라 내레이터/대사 혼재 오디오에서 완전 오매칭.
- **해결**:
  1. faster-whisper로 오디오 전사 → segment + word-level timing 확보
  2. manifest의 bubble text ↔ whisper segment text를 `difflib.SequenceMatcher`로 내용 매칭
  3. 합쳐진 긴 세그먼트도 **word-level sliding window**로 특정 phrase의 실제 시작점 추출
  4. 유사도 0.5 이상 매칭만 채택, 그 외는 유지 (유지 사유 로그)
- **예방**: **silencedetect 단독 사용 영구 금지**. 타이밍 매칭은 반드시 whisper + 내용 매칭.
- **구조적 교훈**: AI는 오디오를 "듣지" 못하므로 텍스트 변환(whisper) + 내용 대조가 유일한 정확한 방법.
- **상태**: ✅ 해결됨 (17건 word-level 재스냅, 4건 유지)

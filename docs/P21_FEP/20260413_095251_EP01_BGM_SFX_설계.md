# EP.01 BGM / SFX 설계서 — 강요된 칭찬의 심리학

## 설계 원칙

1. **BGM은 감정의 밑그림**: 내레이션이 그림이면, BGM은 캔버스의 색감
2. **정적도 연출**: 완전 무음 vs. 미세 앰비언트 구분 — 완전 무음은 "소름" 포인트에만
3. **원형 구조**: 프롤로그 피아노 → 에필로그에서 같은 멜로디 복귀 (감정적 원점 회귀)
4. **대비 극대화**: 코믹 구간 직후 정적, 따뜻한 직후 차가운 — 고저차가 클수록 임팩트

### SFX 소싱 전략
- **무료 라이브러리**: freesound.org, pixabay.com/sound-effects (CC0/Attribution)
- **자체 생성**: 피아노 불협화음(SFX-3)은 GarageBand/Logic에서 직접 녹음 가능
- **AI 생성**: 앰비언트/드론 계열은 Suno/Udio에서 인스트루멘탈 생성 후 구간 추출

## BGM 소싱 전략

### 옵션 1: AI 음악 생성 (권장)
- **Suno AI** 또는 **Udio**: 각 트랙 설명을 프롬프트로 변환
- 장점: 저작권 자유, 감정 커스텀 가능
- 단점: 품질 변동

### 프롬프트 예시
| 트랙 | Suno 프롬프트 방향 |
|------|-------------------|
| BGM-A | "solo piano, minimal, contemplative, gentle, with pauses between notes, Korean drama OST style, 70bpm" |
| BGM-B | "acoustic guitar and light piano, cozy afternoon, slice of life anime BGM, warm, 90bpm" |
| BGM-C | "dark ambient drone, low cello, heartbeat rhythm, psychological tension, minimal, 50bpm" |
| BGM-F | "solo piano, minor key, beautiful but lonely, slow, emotional Korean drama, 60bpm" |
| BGM-G | "most beautiful sad piano, single melody, intimate, like whispering, Korean ballad instrumental, 55bpm" |

### 옵션 2: 로열티프리 라이브러리
- Artlist, Epidemic Sound (유료, 상업 라이선스)
- YouTube Audio Library (무료, 제한적)
- 기존 bgm.mp3 활용 가능 여부 확인

### 옵션 3: 하이브리드
- 메인 테마(BGM-A)는 AI 생성으로 시리즈 아이덴티티 확보
- 나머지는 라이브러리에서 분위기 매칭

## 볼륨 레벨 가이드

| 요소 | 기준 dB | 비고 |
|------|---------|------|
| 내레이터 음성 | 0 dB (기준) | — |
| 캐릭터 대사 | -1 ~ -2 dB | 내레이터보다 약간 낮게 |
| BGM (대사 중) | -18 ~ -22 dB | 대사 방해 금지 |
| BGM (비트/정적) | -12 ~ -15 dB | 정적에서 살짝 올림 |
| BGM (인스트루멘탈 구간) | -8 ~ -10 dB | 대사 없는 감정 구간 |
| SFX | -6 ~ -12 dB | 종류별 상이 |
| 완전 정적 | **-∞** (무음) | S-00, S-07 직후, S-20 직후만 |

## build_video.py 통합 계획

### 오디오 레이어 구조
```
Layer 1: 내레이터 TTS 클립 (씬별 분할)
Layer 2: 캐릭터 대사 TTS 클립 (감정별 분할)
Layer 3: BGM 트랙 (씬별 볼륨 자동화)
Layer 4: SFX (타이밍 포인트 삽입)
Layer 5: 무음 비트 (각본 표기에 따른 정적)
```

### FFmpeg 믹싱 전략
- `amix` 또는 `amerge` 필터로 다중 레이어 합성
- BGM 볼륨 자동화: `volume=enable='between(t,start,end)':volume=0.1` 패턴
- 비트 삽입: `apad` + `atrim`으로 무음 구간 생성

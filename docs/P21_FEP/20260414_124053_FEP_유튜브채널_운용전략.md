# FEP 프로젝트 공통 지침: 유튜브 채널 운용전략

> 이 문서는 `FEP/` 루트에 배치한다.
> 모든 에피소드 작업지시서에서 이 문서를 참조한다.

---

## 1. 업로드 빈도 전략

### 1-1. 핵심 원칙

- **롱폼 주 2~3회** 업로드를 기본 목표로 한다
- **쇼츠는 매일 또는 격일** 업로드하여 채널을 활성 상태로 유지한다
- 에피소드별 쇼츠 흐름이 중첩되는 것은 문제없다. 단, **같은 날 롱폼 2개는 피한다**

### 1-2. 근거

**알고리즘은 개별 영상 중심으로 평가한다.** 유튜브는 채널 평균이 아닌 개별 영상의 클릭률(CTR), 시청 지속 시간, 참여율을 기준으로 추천 여부를 결정한다. 따라서 같은 날 올린 EP.01 쇼츠와 EP.02 쇼츠가 서로의 성과를 깎아먹지 않는다.

**업로드 빈도가 높으면 활성 채널로 인식된다.** 꾸준히 올리는 채널은 알고리즘이 활동적이라 판단하여 노출 기회가 늘어난다. 주 1~2회가 최소 권장 빈도이므로, 주 2~3회 롱폼 + 매일 쇼츠는 이상적인 빈도다.

**쇼츠와 롱폼은 상호 보완적이다.** 2025년 알고리즘 업데이트 이후 쇼츠가 롱폼 조회수를 깎아먹는 현상은 해소되었다. 오히려 쇼츠에서 롱폼으로 유입시키는 구조가 가장 효율적인 성장 전략으로 확인되었다.

**질이 보장되면 양은 무조건 유리하다.** "양보다 질"이라는 원칙은 저품질 양산에 대한 경고이지, 품질 좋은 콘텐츠를 자주 올리는 것을 제한하는 것이 아니다. AI 일러스트 + TTS 파이프라인 특성상 제작 속도를 높일 수 있으므로, 품질을 유지하면서 빈도를 올리는 것이 가능하다.

### 1-3. 롱폼 간격 규칙

- **롱폼 간 최소 2~3일 간격**을 둔다
- 이유: 롱폼 업로드 후 24~48시간이 '골든타임'이다. 이 시기에 초기 조회수·댓글·좋아요가 집중되면 알고리즘이 더 넓은 시청자에게 추천한다. 같은 날 롱폼 2개를 올리면 골든타임이 분산되어 양쪽 모두 초기 성과가 약해진다.
- 예외: 시즌 프리미어, 특별 이벤트 등은 예외적으로 같은 날 가능

## 2. 멀티 에피소드 병행 스케줄

### 2-1. 주 2회 롱폼 모델 (권장)

시즌 1 기준 6편을 주 2회씩 올리면 약 3주에 시즌 완료. 각 에피소드의 쇼츠가 자연스럽게 중첩된다.

**예시 스케줄 (EP.01 수요일 론칭 기준)**

| 요일 | Week 1 | Week 2 |
|------|--------|--------|
| 월 | — | EP.01 심화쇼츠④ + EP.02 티저① |
| 화 | EP.01 티저① | EP.02 티저② |
| 수 | EP.01 티저② | **EP.02 롱폼** |
| 목 | **EP.01 롱폼** | EP.02 심화쇼츠① |
| 금 | EP.01 심화쇼츠① | EP.02 심화쇼츠② |
| 토 | EP.01 심화쇼츠② | **EP.03 롱폼** |
| 일 | EP.01 심화쇼츠③ | EP.02 심화쇼츠③ + EP.03 티저① |

**이 모델의 장점:**
- 롱폼 간 최소 2~3일 간격 유지
- 쇼츠가 매일 1~2개씩 올라가 채널 활성 상태 유지
- 에피소드 간 쇼츠 중첩 시 "이 채널 콘텐츠 많네" → 채널 체류 시간 증가
- 시즌 1 전체를 약 3주 내에 소화 가능

### 2-2. 주 3회 롱폼 모델 (공격적)

제작 파이프라인이 안정화된 이후 적용 가능.

| 요일 | 콘텐츠 |
|------|--------|
| 월 | 이전 EP 심화쇼츠 + 다음 EP 티저 |
| 화 | **롱폼 A** |
| 수 | 롱폼 A 심화쇼츠 |
| 목 | **롱폼 B** |
| 금 | 롱폼 B 심화쇼츠 |
| 토 | **롱폼 C** |
| 일 | 혼합 쇼츠 / 휴식 |

**주의:** 주 3회는 품질 유지가 어려울 수 있다. 초기에는 주 2회로 시작하고, 파이프라인 안정화 후 3회로 확장할 것.

### 2-3. 주 1회 롱폼 모델 (보수적 — 론칭 첫 주)

채널 론칭 첫 주는 EP.01에 집중하여 채널 정체성을 확립한다.

| 요일 | 콘텐츠 | 목적 |
|------|--------|------|
| 월 | EP.01 티저쇼츠 ① | 기대감 조성 |
| 화 | EP.01 티저쇼츠 ② | 호기심 극대화 |
| **수** | **EP.01 롱폼** | 메인 콘텐츠 |
| 목 | EP.01 심화쇼츠 ① | 롱폼 유입 유도 |
| 금 | EP.01 심화쇼츠 ② | 유머 장면 |
| 토 | EP.01 심화쇼츠 ③ | 심리학 키워드 유입 |
| 일 | EP.01 심화쇼츠 ④ | 공감 포인트 |
| 월 | EP.01 심화쇼츠 ⑤ + EP.02 티저① | 다음 에피소드 예고 |

론칭 2주차부터 주 2회 모델로 전환.

## 3. 쇼츠 운용 원칙

### 3-1. 쇼츠의 역할

쇼츠는 세 가지 역할을 수행한다:

| 역할 | 유형 | 게시 시점 |
|------|------|-----------|
| **예고** | 티저 쇼츠 (2편) | 롱폼 D-2, D-1 |
| **심화** | 심화 쇼츠 (5편) | 롱폼 D+1 ~ D+5 |
| **채널 활성화** | 위 두 가지의 중첩 | 상시 |

### 3-2. 같은 날 쇼츠 2개 가능 조건

- 서로 다른 에피소드의 쇼츠여야 한다 (EP.01 심화 + EP.02 티저)
- 같은 에피소드의 쇼츠 2개를 같은 날 올리는 것은 피한다
- 게시 시간을 분리한다 (오후 2시 + 저녁 8시)

### 3-3. 쇼츠 포스팅 시간 (한국 기준)

- 평일: 오후 2~3시 또는 저녁 8~9시
- 주말: 오전 11시 또는 저녁 7~8시
- 롱폼과 같은 날이면 쇼츠를 먼저 올리고, 2~3시간 뒤 롱폼 게시

## 4. 알고리즘 대응 전략

### 4-1. 골든타임 활용

업로드 직후 24~48시간이 골든타임이다. 이 시기에 초기 반응을 극대화하는 방법:

- **업로드 시간**: 시청자 활동이 많은 시간대에 맞춤 (YouTube Studio 데이터 참고)
- **커뮤니티 탭**: 롱폼 업로드 직전/직후 커뮤니티 게시물로 알림
- **SNS 연동**: 업로드 직후 트위터/인스타 등에 공유
- **댓글 소통**: 업로드 후 1~2시간 내 들어오는 댓글에 적극 응답
- **알림 설정 유도**: 영상 내 CTA에서 알림 설정 언급

### 4-2. 영상 간 연결

- **엔드스크린**: 롱폼 끝에 다음 에피소드 또는 관련 쇼츠 링크
- **카드**: 영상 중간 관련 개념이 언급될 때 해당 에피소드 카드 삽입
- **재생목록**: "시즌 1 전체", "심리학 개념별", "쇼츠 모음" 등 재생목록 구성
- **쇼츠 → 롱폼 링크**: 쇼츠 설명란에 관련 롱폼 링크 필수

### 4-3. 채널 체류 시간 극대화

에피소드 간 쇼츠가 중첩되면 시청자가 여러 영상을 연속 시청할 가능성이 높아진다. 이는 세션 시간(session time) 증가로 이어지고, 알고리즘이 채널 전체를 우호적으로 평가한다.

### 4-4. 커뮤니티 탭 활용

영상을 올리지 않는 날에도 커뮤니티 탭을 활용하면 알고리즘이 채널을 활발하다고 인식한다.

활용 예시:
- 투표: "다음 에피소드 주제 뭐가 좋아요?" → 참여 유도
- 밈/짤: 에피소드 관련 한 컷 일러스트 + 짧은 텍스트
- 비하인드: "이번 에피소드 만들면서 가장 놀랐던 심리학 사실"
- 질문: "여러분은 읽씹하는 편인가요, 당하는 편인가요?"

## 5. 채널 성장 단계별 전략

### Phase 1: 론칭기 (EP.01~02, 약 2주)

- 주 1회 롱폼 + 매일 쇼츠
- EP.01에 채널 정체성을 집중 — "이 채널이 뭘 하는 곳인지" 확립
- 쇼츠를 통해 쇼츠 피드 노출 → 채널 인지도 확보
- 커뮤니티 탭 투표로 시청자 참여 시작

### Phase 2: 확장기 (EP.03~06, 약 3주)

- 주 2회 롱폼으로 전환
- 에피소드 간 쇼츠 중첩 시작
- 재생목록 구성하여 연속 시청 유도
- 시청자 댓글에서 에피소드 아이디어 수집 → 시즌 2 준비

### Phase 3: 안정기 (시즌 2~)

- 주 2~3회 롱폼
- 파이프라인 최적화로 제작 속도 안정화
- 시청자 제보 에피소드 도입 → 참여형 콘텐츠
- 조회수/구독자 데이터 기반으로 업로드 요일·시간 최적화

## 6. SEO 및 검색 노출

### 6-1. 제목 최적화

- "___의 심리학" 구조 유지 — 채널 브랜드와 통일
- 부제에 검색 가능한 구체적 키워드 포함 ("나 예쁘지?", "읽씹", "치킨 마지막 한 조각")
- 길이: 40~60자 (모바일에서 잘리지 않는 범위)

### 6-2. 설명란 구성

```
[필수 요소]
- 영상 요약 (2~3문장)
- 타임스탬프 (챕터) — 구글 검색 '주요 장면' 노출에 필수
- 관련 에피소드 링크
- 채널 구독 링크
- SNS 링크

[예시]
"나 예쁘지?" — 이 다섯 글자 안에 숨어 있는 세 가지 심리학적 구조를 분석합니다.

⏱ 타임스탬프
00:00 인트로
00:15 확인 추구 행동
01:45 강요된 동의
...

📌 관련 에피소드
▶ EP.02 읽씹의 심리학: [링크]
```

### 6-3. 태그 전략

- 핵심 키워드: 에피소드별 심리학 용어 (인지부조화, 거부 민감성 등)
- 일상 키워드: 에피소드별 상황어 (읽씹, 나 예쁘지, 치킨 마지막 등)
- 채널 키워드: 일상속 경험의 심리학, 일상 심리학, 심리학 해설
- 경쟁 키워드: 비치키, 심리학 유튜브, 심리분석

### 6-4. 썸네일 A/B 테스트

- 업로드 후 48시간 내 CTR 확인
- CTR이 4% 미만이면 썸네일 교체 고려
- 키워드 태그 배치, 텍스트 크기, 캐릭터 표정 등을 변수로 테스트

## 7. 콘텐츠 간 시너지 설계

### 7-1. 에피소드 간 개념 연결

같은 심리학 개념이 여러 에피소드에 등장할 때, 영상 내에서 자연스럽게 교차 참조한다.

| 개념 | 등장 에피소드 | 연결 방법 |
|------|-------------|-----------|
| 인지부조화 | EP.01, EP.03 | EP.03에서 "EP.01에서 다뤘던 인지부조화, 기억나시죠?" |
| 도구화 | EP.01, EP.04 | EP.04에서 "칭찬 자판기 기억나시죠? 이번엔 관계 전체가 도구화됩니다" |
| 사회적 바람직성 | EP.01, EP.03, EP.05 | 시즌 관통 테마로 활용 |
| 방관자 효과 | EP.02, EP.03 | EP.03에서 "단톡 읽씹과 같은 구조입니다" |

### 7-2. 쇼츠 → 롱폼 유입 퍼널

```
[쇼츠 시청자]
    ↓ "전체 분석은 프로필 영상에서"
[롱폼 시청]
    ↓ 엔드스크린 "다음 에피소드"
[시리즈 연속 시청]
    ↓ 재생목록 자동 재생
[채널 구독]
    ↓ 알림 설정
[충성 시청자]
```

## 8. 수익화 고려사항

### 8-1. 수익 창출 자격 조건

- 구독자 1,000명 + 최근 12개월 공개 시청 시간 4,000시간
- 또는 구독자 1,000명 + 최근 90일 쇼츠 조회수 1,000만회
- 채널 6개월 이상 비활성 시 수익 창출 자격 박탈 가능

### 8-2. 초기 수익화 전략

- 쇼츠 조회수 경로가 더 빠를 수 있음 (매일 쇼츠 업로드 시)
- 롱폼은 광고 수익 단가(CPM)가 쇼츠보다 높음
- 양쪽 모두 병행하여 조건 달성을 가속화

### 8-3. 광고 친화적 콘텐츠

- 심리학 교양 콘텐츠는 광고주 친화적 카테고리
- 과도한 자극, 욕설, 논란성 표현 회피
- 채널 톤 "장난스럽지만 핵심은 정확하게"가 광고 적합성과 일치

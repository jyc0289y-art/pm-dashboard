## 4. 銭湯 (센토) 문화 딥다이브

### 아오모리시 센토의 특이점

> **아오모리시의 센토 = 천연 온천**. 시내 15곳 중 10곳이 자가 온천원을 보유.
> 쇼와 45년(1970) 62곳에서 현재 15곳으로 감소했지만, 생존한 곳들은 모두 진짜 온천.
> **입욕료 전 점포 통일: 대인 450엔 / 중인 150엔 / 소인 60엔**

### 추천 센토 3선

#### (1) 出町温泉 (데마치온천)
- **주소**: 아오모리역/신아오모리역 모두 차량 9분 거리 (주택가)
- **영업시간**: 6:00~23:00
- **요금**: 450엔
- **천질**: 알칼리성 단순천 (pH 9.2) -- 원천 그대로 가열 없이 제공
- **특징**: 1969년부터 이어진 후지산 벽화(背景画). **아침 6시 오픈**으로 아오모리의 "아침 목욕 문화" 체험 가능.
- **왜 특별한가**: 수도꼭지에서 나오는 물이 온천 원수 그 자체. 관광객이 거의 없는 완전 로컬.

#### (2) 徳乃湯温泉 (토쿠노유온천)
- **주소**: 筒井駅 (츠츠이역, 아오이모리 철도) 도보 2분
- **영업시간**: 11:30~22:00 (일요일 9:30~)
- **요금**: 450엔
- **천질**: 원천 온도 27.8도C
- **특징**: **도와다호+오이라세 계류 벽화**가 압권. 열차로 접근 가능한 유일한 역세권 센토. 사우나, 냉탕 완비.

#### (3) 大福温泉 (다이후쿠온천)
- **주소**: 아오모리역 차량 15분 (주택가)
- **영업시간**: 12:00~22:00 (일요일 8:30~)
- **요금**: 450엔
- **천질**: 알칼리성 단순천 (pH 8.64) -- "미인수"로 불림
- **특징**: 다람쥐/곰 모양의 물 토출구가 귀여움. 사우나, 냉탕, 좌욕, 와탕 등 욕조 종류 다양. 자극이 적어 매일 이용에 최적.

### 센토 방문 팁
- **수건/비누 지참 필수** (판매도 하지만 200엔 정도 추가)
- **문신 OK인 곳이 대부분** (일반 센토는 슈퍼센토보다 관대)
- **아침 목욕(朝風呂)**: 출마치온천 6:00, 다이후쿠온천 일요일 8:30 등 새벽/아침 영업하는 곳 다수

## 6. 地元パン屋 (현지 빵집)

### 工藤パン (Kudo Pan) -- 아오모리의 빵 왕국

- **창업**: 1932년 むつ市의 작은 빵집에서 시작
- **대표 상품**: イギリストースト (1967년 발매, 월 50만개 판매)
- **유통**: 편의점, 슈퍼마켓 전체에 유통 (전용 매장이 아닌 유통형)
- **공식 사이트**: https://www.kudopan.co.jp/

#### イギリストースト 현재 8종 바리에이션

| 종류 | 설명 |
|------|------|
| **クラシック (오리지널)** | 마가린 + 그래뉼러 슈가. 원조의 맛. |
| **つぶ入りピーナッツクリーム** | 피넛 크림 버전 |
| **りんごジャム＆北海道練乳クリーム** | 아오모리산 사과잼 + 홋카이도 연유크림. **아오모리 한정감 최고** |
| **チョコスプレー＆クリーム** | 초코 스프링클 + 크림 |
| **ジャリましブレンドコーヒー** | 커피맛, 더 바삭한 슈가 |
| **ジャリジャリMAX** | 슈가 코팅 극대화 버전 |
| **もっとジャリまし** | 중간 레벨 바삭함 |
| **ブレンドコーヒー** | 커피맛 베이직 |

> **"ジャリジャリ"**: 그래뉼러 슈가를 씹을 때 나는 바삭한 식감을 표현하는 의태어. 이 식감이 イギリストースト의 핵심.

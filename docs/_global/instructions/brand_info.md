# 회사 및 브랜드 정보

## SL Corporation
- **정식명칭**: SL Corporation
- **서비스 브랜드**: SeouLink (SL)
- **브랜드 컨셉**: "서울(Seoul)"과 "영혼(Soul)"의 중의성 + "연결(Link)"의 의미
- **타겟 고객**: 한국에 관심 있는 글로벌 고객
- **서비스 영역**: 여행, 어학 교육, 한국인과의 교류 활동
- **브랜드 네이밍 규칙**: 개별 서비스명 뒤에 "SL"을 붙여 브랜드 통일성 확보 (예: `서비스명 SL`)

## 진행 중인 프로젝트
- **SeouLink 한국어교실**: "안녕, 서울!" 교재 기반 한국어 교육
- **LingoPlay SL** (가칭, 서비스명 미확정): YouTube 자막 기반 다국어 영상 학습 앱 (기획 완료, 개발 준비). 이 이름은 해당 프로젝트 세션에서만 사용 중인 가칭이며, 전사 확정 명칭이 아님
- **TripLink SL**: 글로벌 여행 가이드 플랫폼 (기획 시작). 다양한 여행지 콘텐츠를 다국어로 제공. 첫 번째 테마: 일본 소도시 나이트라이프 가이드 (15개 도시). 기존 여행 가이드 HTML/데이터를 앱 콘텐츠로 전환 예정
- 한국인 교류 활동 서비스 (기획 예정)

## 운영 규칙

1. **등록**: 재사용 가능한 독립 모듈 완성 시 즉시 등록
2. **버전 관리**: 모듈 업데이트 시 버전 bump + 상세 카드 갱신 + 변경 이력 추가
3. **포크 이력**: 어떤 프로젝트가 어떤 버전을 포크했는지 추적
4. **번호**: 전역 순번 (MOD-001, MOD-002, ...), 프로젝트 무관

# HighLink 로컬 MVP 테스트 결과 보고서

> **프로젝트**: P18 HighLink — 고속도로 AI 관제 시스템  
> **세션**: P18.7.2.0  
> **작성일**: 2026-04-16  
> **테스트 환경**: macOS / Apple M5 / 32GB RAM / MPS (Metal Performance Shaders)

---

## 1. 테스트 개요

### 1.1 목적

Phase 1~7 전체 구현 후, F1~F4 핵심 기능의 실전 동작을 검증한다.

### 1.2 테스트 범위

| 기능 | 설명 | 테스트 항목 |
|------|------|-----------|
| **F1** | 양방향 통행 인식 | 자동 캘리브레이션 (벡터 수집 → 정상 방향 결정) |
| **F2** | 역주행 감지 | 정상 방향 대비 이탈 각도 → 이벤트 발생 |
| **F3** | 보행자/이륜차 인지 | 고속도로 내 person/bicycle/motorcycle 감지 |
| **F4** | 정체/사고 인지 | 차량 급정지 + 차로 정차 + 회피 기동 감지 |
| **F5** | 악천후 대응 | WeatherAdapter 파라미터 동적 조정 |
| **F6** | 이벤트 알림 | 3단계 알림 (CRITICAL/WARNING/INFO) |

### 1.3 테스트 환경

| 항목 | 사양 |
|------|------|
| **하드웨어** | Apple M5 SoC, 32GB 통합 메모리 |
| **OS** | macOS |
| **Python** | 3.12.6 |
| **AI 모델** | YOLO11L (Ultralytics) |
| **추적기** | ByteTrack (persist=True) |
| **추론 디바이스** | MPS (Metal Performance Shaders) |
| **입력 해상도** | 640×360 (테스트 영상) |
| **추론 해상도** | 640×640 (imgsz=640) |

### 3.5 F5 악천후 대응 — 기능 테스트

```python
# WeatherAdapter 단위 테스트 결과
안개 조건 (fog + very_poor visibility):
  conf_threshold: 0.40 → 0.25  (Δ-0.15)
  iou_threshold: 0.50 → 0.50  (변동 없음)
  context_label: "[안개] "
  
폭우 조건 (heavy_rain):
  conf_threshold: 0.40 → 0.30  (Δ-0.10)
  iou_threshold: 0.50 → 0.55  (Δ+0.05)
  context_label: "[폭우] "
```

| 프리셋 | conf 조정 | iou 조정 | 적용 근거 |
|--------|----------|----------|----------|
| clear | 0.00 | 0.00 | 기본값 유지 |
| rain | -0.05 | +0.03 | recall 우선, 물방울 오검출 방지 |
| heavy_rain | -0.10 | +0.05 | 강한 recall, 강한 오검출 방지 |
| snow | -0.05 | +0.03 | 눈발 오검출 방지 |
| fog | -0.10 | 0.00 | 가시거리 저하 대응 |
| mixed | -0.08 | +0.03 | 복합 악천후 |
| night | -0.05 | 0.00 | 야간 대응 |

## 배경

이전 데모 영상에서 SUV/밴이 "트럭"으로 오분류되는 문제가 있었습니다.
COCO 데이터셋에 SUV/밴 클래스가 없어 큰 승용차가 truck으로 분류되기 때문입니다.

**이번 커밋에서 후처리 휴리스틱이 추가되었습니다:**

`engine/detection/detector.py` → `reclassify_truck()` 함수:
- truck(7)으로 감지된 객체의 **bbox 종횡비 < 1.8** AND **상대 면적 < 5%** 이면 car(2)로 재분류
- 진짜 트럭(가로로 긴 대형 차량)은 그대로 truck 유지
- `scripts/generate_demo_video.py`와 `app/streamlit/app.py`에도 동일 로직 적용됨

## 4단계: 재분류 효과 검증

v1과 v2를 비교하여 truck → car 재분류 건수를 확인합니다.

```python
import cv2, sys
sys.path.insert(0, '.')
from ultralytics import YOLO
from engine.detection.detector import reclassify_truck

model = YOLO("yolo11l.pt")
cap = cv2.VideoCapture("data/samples/mixed_traffic.mp4")

reclassified_count = 0
total_truck_before = 0
total_truck_after = 0

for i in range(100):
    ret, frame = cap.read()
    if not ret: break
    h, w = frame.shape[:2]
    results = model.predict(frame, conf=0.4, classes=[0,1,2,3,5,7], device="cuda:0", verbose=False)
    for box in results[0].boxes:
        cls_id = int(box.cls.item())
        if cls_id == 7:
            total_truck_before += 1
        bbox = box.xyxy[0].cpu().numpy().tolist()
        new_cls = reclassify_truck(bbox, cls_id, h, w)
        if new_cls == 7:
            total_truck_after += 1
        if cls_id != new_cls:
            reclassified_count += 1

cap.release()
print(f"truck 변화: {total_truck_before} → {total_truck_after} (재분류: {reclassified_count}건)")
```

## 5. 특이사항

- EfficientNet-B0 추가에 따른 FPS 영향이 거의 없음 (31MB GPU 메모리, <0.5fps 감소)
- ffmpeg 미설치로 mp4v 코덱 사용 중. H.264 인코딩을 위해 ffmpeg 설치 권장
- 맥북에서 모드C 웹캠 버그를 직접 수정 완료 (커밋 e6bd5fe)
- Cloudflare Tunnel URL은 Quick Tunnel이므로 서버 재시작 시 변경됨
- 비밀번호: secrets.toml에 저장됨 (highlink2026)

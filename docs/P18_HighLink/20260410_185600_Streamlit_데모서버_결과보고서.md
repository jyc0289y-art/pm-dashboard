## 5. 특이사항

1. **winget 설치 실패**: `winget install cloudflared`가 대화형 승인 필요로 실패하여 GitHub에서 바이너리 직접 다운로드
2. **ffmpeg 미설치**: 출력 영상이 mp4v 코덱으로 저장됨. 일부 브라우저에서 재생 불가할 수 있으나 Streamlit 내장 플레이어에서는 정상
3. **Quick Tunnel URL 일시적**: PC 재시작 또는 cloudflared 재실행 시 URL 변경됨. 고정 URL 필요 시 Cloudflare 계정 등록 후 Named Tunnel 설정 필요
4. **ITS API 키 미설정**: 실시간 CCTV 모드 테스트 불가. API 키 설정 후 재검증 필요
5. **비밀번호 인증**: `st.secrets` 기반. `.streamlit/secrets.toml`에서 변경 가능
6. **절전 모드**: PC 절전 비활성화 필요 (장시간 시연 시)

# HighLink — Streamlit 데모 서버 구축 작업지시서

> **이 문서는 회사 PC(RTX 4060)의 Claude Code가 읽고 실행하기 위한 자동화 지시서입니다.**
> 모든 단계를 순서대로 실행하고, 마지막에 결과보고서를 지정된 양식으로 생성해 주세요.

## 배경

HighLink 고속도로 AI 관제 시스템의 데모를 사업 파트너에게 시연하기 위해,
이 PC를 Streamlit 서버로 구성합니다.

**핵심 요구사항:**
- 파트너는 **브라우저 접속만** 하며, 소스코드에 일절 접근하지 않음
- Cloudflare Tunnel로 외부 HTTPS 접속 제공
- RTX 4060에서 YOLO11 추론 (28+ FPS)
- ITS CCTV API로 실시간 고속도로 영상 감지 시연

---

## ⚠️ 보안 필수 준수사항

**아래 규칙은 모든 작업 단계에서 반드시 지켜야 합니다.**

### API 키 관리
- API 키는 **`.streamlit/secrets.toml`에만 저장** (Git 미포함, `.gitignore` 적용됨)
- **절대 코드 파일(.py)에 하드코딩 금지**
- **절대 커밋 메시지, 결과보고서, 로그 파일에 키값 포함 금지**

### Git 커밋 시
- `git add .` 또는 `git add -A` **사용 금지** → 파일을 명시적으로 지정하여 add
- 커밋 전 `git status`로 `.streamlit/secrets.toml`, `.env`가 스테이징에 없는지 확인

### 외부 접속
- Cloudflare Tunnel URL을 공개 채널에 게시 금지 → 파트너에게만 DM 전달
- 반드시 **비밀번호 인증 활성화 상태**에서 Tunnel 실행
- 사용하지 않는 Tunnel은 즉시 종료

### 소스코드 보호
- 파트너에게 SSH/원격 데스크톱 접속 제공 금지
- 에러 메시지에 경로/코드가 노출되지 않도록 운영 시:
  ```bash
  streamlit run app/streamlit/app.py --server.port 8501 --logger.level error
  ```

## 3단계: 한국어 폰트 경로 확인

Streamlit 앱은 OS별 폰트를 자동 감지하지만, 확인이 필요합니다:

```python
from pathlib import Path
fonts = [
    "C:/Windows/Fonts/malgun.ttf",
    "C:/Windows/Fonts/NanumGothic.ttf",
    "C:/Windows/Fonts/gulim.ttc",
]
found = [f for f in fonts if Path(f).exists()]
print(f"사용 가능한 한국어 폰트: {found}")
```

`malgun.ttf`(맑은 고딕)가 있으면 자동으로 사용됩니다. 없으면 `app/streamlit/app.py`의 `FONT_CANDIDATES` 리스트에 발견된 경로를 추가하세요.

## 4단계: API 키 설정 (secrets.toml)

**환경변수가 아닌 `.streamlit/secrets.toml`에 저장합니다.**
이 파일은 `.gitignore`에 포함되어 있어 Git에 커밋되지 않습니다.

```bash
# .streamlit 디렉토리 생성
mkdir -p .streamlit
```

`.streamlit/secrets.toml` 파일을 아래 내용으로 생성:

```toml
password = "highlink2026"
its_api_key = "여기에_ITS_API_키_입력"
```

**⚠️ 이 파일을 절대 Git에 커밋하지 마세요. `.gitignore`가 정상 적용되었는지 확인:**
```bash
git status
# .streamlit/secrets.toml이 목록에 나타나면 안 됨
```

**API 키 동작 확인:**
```python
import streamlit as st
# 또는 직접 테스트:
import toml
secrets = toml.load(".streamlit/secrets.toml")
print(f"API 키 설정됨: {secrets['its_api_key'][:8]}...")

from engine.ingest.cctv_client import ITSCCTVClient
client = ITSCCTVClient(secrets["its_api_key"])
cctvs = client.get_highway_cctvs()
print(f"고속도로 CCTV {len(cctvs)}개 조회 완료")
```

## 6단계: Cloudflare Tunnel 설치

### 6-1. cloudflared 설치

```powershell
# Windows (winget)
winget install --id Cloudflare.cloudflared

# 또는 직접 다운로드: https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/downloads/
```

### 6-2. 빠른 터널 (도메인 없이 즉시 사용)

```bash
cloudflared tunnel --url http://localhost:8501
```

출력 예시:
```
Your quick Tunnel has been created! Visit it at (it may take some time to be reachable):
https://random-name-here.trycloudflare.com
```

이 URL을 파트너에게 공유하면 됩니다.

### 6-3. 고정 도메인 터널 (선택 — Cloudflare 계정 필요)

```bash
# 로그인
cloudflared tunnel login

# 터널 생성
cloudflared tunnel create highlink

# 설정 파일 생성 (~/.cloudflared/config.yml)
# tunnel: <터널-UUID>
# credentials-file: ~/.cloudflared/<터널-UUID>.json
# ingress:
#   - hostname: highlink.yourdomain.com
#     service: http://localhost:8501
#   - service: http_status:404

# DNS 레코드 추가
cloudflared tunnel route dns highlink highlink.yourdomain.com

# 실행
cloudflared tunnel run highlink
```

## 7단계: 보안 설정

### 7-1. Streamlit 비밀번호 (간단)

`app/streamlit/app.py` 맨 위 (st.set_page_config 다음)에 아래 코드를 추가:

```python
# ── 비밀번호 인증 ──
def check_password():
    if "authenticated" not in st.session_state:
        st.session_state.authenticated = False
    if st.session_state.authenticated:
        return True
    password = st.text_input("비밀번호를 입력하세요", type="password")
    if password == st.secrets.get("password", "highlink2026"):
        st.session_state.authenticated = True
        st.rerun()
    elif password:
        st.error("비밀번호가 틀렸습니다")
    return False

if not check_password():
    st.stop()
```

비밀번호 변경: `.streamlit/secrets.toml` 파일 생성
```toml
password = "원하는_비밀번호"
```

### 7-2. Cloudflare Access (고급 — 선택)

Cloudflare Zero Trust 대시보드에서:
1. Access > Applications > Add
2. Self-hosted, URL: `highlink.yourdomain.com`
3. Policy: 이메일 OTP 인증 (파트너 이메일 등록)
4. 무료 플랜으로 50명까지 지원

## 7. 특이사항

1. **영상 선정에 4회 시도 필요**: 의뢰서의 후보 영상 3개 모두 차종 다양성 부족(2종 이하). YouTube에서 한국 도로 영상을 추가 탐색하여 "South Korea Traffic 4K UHD" (Opluy0Aihis) 선정
2. **ffmpeg 미설치**: mp4v 코덱으로 저장됨. 일부 플레이어에서 재생 불가할 수 있음. ffmpeg 설치 후 H.264 변환 권장
3. **300프레임 데모에서는 3종만 감지**: 영상 초반부에 승용차/보행자/트럭 위주. 버스/자전거/오토바이는 영상 중후반부에 등장하여 **전체 영상(2946프레임)에서 6종 전부 감지**
4. **YOLO11S 경량 모델**: 59.6fps로 매우 빠르지만, 3종만 감지 (소형 객체 놓침). 운영 환경에서는 YOLO11L 권장
5. **FONT_PATH를 Windows 경로로 변경**: `C:/Windows/Fonts/malgun.ttf` (맑은 고딕). macOS 전용 경로에서 변경함
6. **full 영상 57MB**: GitHub 100MB 제한 이내이나, Git LFS 미사용 시 레포 크기 증가에 주의

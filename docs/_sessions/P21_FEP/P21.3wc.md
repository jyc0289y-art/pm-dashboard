## 기술 발견사항

### LivePortrait pkl 템플릿 호환성
- 이름있는 템플릿 (wink.pkl, aggrieved.pkl 등): motion 데이터만 포함, c_d_eyes_lst/c_d_lip_lst 없음
- 번호 템플릿 (d0.pkl, d1.pkl 등): 구버전 키 포함
- 해결: pipeline.py에 zeros 폴백 패치

### zoompan + 비디오 입력 비호환
- zoompan 필터는 비디오 입력에서 "Failed to configure output pad" 에러 발생
- 해결: scale+crop 조합으로 미세 줌 구현 (약간 크게 스케일 후 사인파 crop 이동)

### 빈 말풍선 프로그래밍적 제거 한계
- 흰색 내부만 있는 말풍선: scipy.ndimage로 감지 + 배경색 페인팅으로 제거 가능
- 어두운 외곽선이 있는 말풍선: 캐릭터 외곽선과 구별 불가 → 사각 영역 블랭킷 페인팅으로 해결
- 일러스트 선화 스타일로 그려진 말풍선(scene_11): 프로그래밍적 제거 불가 → ComfyUI 재생성 필요
- 근본 해법: AI 일러스트 생성 프롬프트에서 "no speech bubbles" 강화

## 금지/확정 사항
- ❌ LivePortrait pkl 템플릿 중 named templates는 eye/lip 데이터 없음 — pipeline.py 패치 필수
- ❌ FFmpeg zoompan 필터는 비디오 입력과 호환 불가 — scale+crop 조합 사용
- ❌ LivePortrait는 단순한 만화체 얼굴 미감지 — 해당 씬은 브리딩 폴백
- ❌ B컷 세그먼트에 LP 적용 금지 — B컷은 별도 이미지이므로 메인 씬 LP 영상을 사용하면 안됨
- ❌ `--overlay` 빌드 시 원본 덮어쓰기 금지 — `_ko` 접미사 자동 적용 (build_video.py 수정 완료)
- ❌ 쇼츠에서 하드 9:16 크롭 금지 — 블러 배경 레터박스 방식 사용
- ✅ LivePortrait 배치: 14/15 성공, scene당 ~40-60초 (MPS)
- ✅ LP 클립 빌드: -stream_loop -1 + scale+crop(미세줌) + -t duration
- ✅ LP 브리딩 강도: BREATHING_INTENSITY × 0.4 (LP 위의 브리딩은 약하게)
- ✅ 교훈집 HTML: 20260412_232351_EP01_교훈집_시각화.html
- ✅ 썸네일: scene_03 기반, 1280×720, 그라디언트 오버레이 + 제목 텍스트

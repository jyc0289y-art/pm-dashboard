# P10.6wc — 서드파티 소프트웨어 아이템 발굴 리서치

## 작업 요약
Hodlum(Ludlum 3030-2 서드파티 소프트웨어) 개발 경험을 토대로, 대중적 수요가 큰 서드파티 소프트웨어 아이템을 발굴하는 시장 조사를 수행. 최종적으로 **게이밍 RGB 주변기기 통합 컨트롤러**를 1순위 아이템으로 선정하고, 기능 정의·테스트 장비 전략·배포/라이선스 구조까지 설계 완료.

## 주요 결과물

### 1. 선정 아이템: RGB 통합 컨트롤러
- **시장 규모**: 게이밍 액세서리 $130억(2025) → $231억(2031), CAGR 10%
- **경쟁사**: OpenRGB(오픈소스, GUI 최악 불만율 32%), SignalRGB(Windows 전용, 구독 $5/월, 3.5GB bloat)
- **진입 기회**: macOS 지원, 모바일 리모컨, 모던 UI, 경량화, 일회성 결제

### 2. 기능 로드맵
- **MVP (v1.0)**: USB HID 기기 제어, 모던 GUI(HOBIS 기반), 기기 자동 감지, 색상/이펙트 제어, 프로필 저장, Windows + macOS
- **v1.5**: 모바일 리모컨(iOS/Android), SMBus/I2C(RAM, 마더보드), GPU RGB
- **v2.0**: 오디오 비주얼라이저, 스크린 앰비언스, 플러그인 시스템

### 3. 테스트 장비 전략
- **Phase 1** (~$100): Corsair 키보드 1개 → 프로토콜 파싱 + GUI 개발
- **Phase 2** (~$80): Razer 마우스 + Redragon 저가 세트 → 멀티 벤더 호환성
- **Phase 3** (~$150): RGB RAM + LED 스트립 + 팬 → SMBus 확장
- **총 예산**: ~$330 (전량 개발 후 처분 가능)

### 4. 배포 & 라이선스 구조
- **중앙 계정 시스템**: 이메일 기반, 어느 스토어에서 구매해도 동일 라이선스
- **스토어 브릿지**: Steam 구매 → 중앙 계정 연동 → macOS/다른 플랫폼에서도 사용
- **결제**: Lemon Squeezy (자체사이트, 5%+50¢) + Steam DRM 브릿지
- **배포 순서**: 자체사이트 → Steam ($100 등록비) → Microsoft Store

### 5. 비즈니스 모델 (최종 확정)
- **원칙**: 원타임 결제 + 업그레이드팩 추가결제 모델
- **Free**: 기기 제어(5대), 기본 이펙트 10종, 프로필 3개
- **Pro ($9.99 일회성)**: 무제한 기기, 50+ 이펙트, 크로스플랫폼, 모바일 리모컨
- **Pro+ ($19.99 일회성)**: 오디오 비주얼라이저, 스크린 앰비언스, 게임 연동
- **업그레이드팩 (추가 유료 DLC)**: 신규 이펙트 팩, 게임 연동 확장, 테마 팩 등 → 지속 수익 + 구독 피로 없이 가치 기반 지출 유도
- **차별점**: SignalRGB 연간 $60 구독 vs 우리 $9.99 원타임 + 선택적 업그레이드팩

### 6. 기술 스택 (Hodlum 재활용)
- PyQt6 GUI + HOBIS Design System
- USB HID 통신 (hidapi/libusb) ← Hodlum 시리얼 통신 경험
- 실시간 데이터 파싱 + 시각화
- 크로스플랫폼 빌드 (PyInstaller/Nuitka)
- OpenRGB 오픈소스 프로토콜 참조 (GPL-2.0)

## 핵심 결정사항
1. **1순위 아이템**: 게이밍 RGB 주변기기 통합 컨트롤러 → 확정
2. **비즈니스 모델**: 구독제 거부, 원타임 결제 + 업그레이드팩 DLC → 확정
3. **크로스 스토어 라이선스**: 중앙 계정 기반, 어디서 사든 모든 플랫폼 사용 가능 → 확정
4. **기술 스택**: Hodlum과 동일 (PyQt6 + USB 통신) → 확정
5. **1차 타겟 벤더**: Corsair (프로토콜 가장 잘 문서화, iCUE bloatware 불만 최다)

## 차기 아이템 후보 (미래 참조용)

### 2순위: 3D 프린터 통합 모니터링 대시보드
- OctoPrint은 Raspberry Pi 필요 → 데스크톱 네이티브 수요 존재
- 시리얼 통신(G-code), 실시간 그래프(온도/진행률) → Hodlum과 거의 동일 구조
- 멀티 프린터 관리, AI 실패 감지, slicer 프리뷰 통합이 차별화 포인트
- 3D 프린터 유저 급증 중, 교육/메이커 시장

### 3순위: 범용 계측기 데이터 로거
- 멀티미터/오실로스코프/환경 센서 등 저가 USB 계측기 전반
- Hodlum 코드 거의 그대로 재활용 가능
- 교육/메이커/소규모 랩 니치 + 충성도 높은 유저

### 4순위: 개인 금융 대시보드 (로컬 퍼스트)
- Reddit 결제 의사 전 카테고리 1위 (193건)
- 프라이버시 수요 급증 (구독 피로 ~7%)
- 리스크: 금융 데이터 보안 책임, 은행 API 연동 복잡성

### 기타 주목할 만한 수요
- 광고 없는 레시피 앱 (Reddit 감정 강도 높음)
- ADHD 워크플로우 도구 (r/ADHD 커뮤니티 구체적 기능 요청 폭주)
- AI 음성녹음기 서드파티 (Plaud Note 등 구독 강제 → 로컬 Whisper 대체)

## 총평 — Hodlum 경험의 사업적 확장: 서드파티 소프트웨어 시장 진입 전략

### 이번 세션의 의의

P10.6wc는 Hodlum(Ludlum 3030-2 전용 도구)의 개발 경험을 **범용 사업 기회**로 전환하려는 전략 세션이다. 대중적 수요가 큰 서드파티 소프트웨어 아이템을 체계적으로 발굴하고, RGB 통합 컨트롤러를 1순위로 선정하여 기능 정의, 비즈니스 모델, 테스트 장비 전략, 배포/라이선스 구조까지 설계했다.

핵심 비즈니스 결정은 **"구독제 거부 + 원타임 결제 + 업그레이드팩 DLC"** 모델이다. SignalRGB의 연간 $60 구독에 대한 사용자 불만을 정면으로 공략하여, $9.99 일회성 결제 + 선택적 DLC라는 차별화된 가격 구조를 설계했다. 중앙 계정 기반 크로스 스토어 라이선스(어디서 구매해도 모든 플랫폼 사용 가능)도 사용자 친화적 결정이다.

기술적으로는 Hodlum과 동일한 스택(PyQt6 + HOBIS + USB 통신)을 재활용하여 개발 비용을 최소화하는 전략이 돋보인다. OpenRGB의 GPL-2.0을 회피하면서도 프로토콜 지식을 참조하는 방법론(네트워크 SDK 연동 또는 클린룸 구현)도 법적 리스크를 관리하는 실용적 접근이다.

2~4순위 후보(3D 프린터 모니터링, 범용 계측기 로거, 개인 금융 대시보드)까지 정리한 것은 1순위가 실패해도 피벗할 수 있는 포트폴리오를 확보한 것이다.

### 전략적 제언

**단기**: OpenRGB 소스코드 분석(P10.7wc)과 Corsair 키보드 구매로 프로토콜 실험을 시작하는 것이 MVP 착수의 첫 단계다.

**중기**: Lemon Squeezy 계정 생성과 라이선스 키 시스템 프로토타입을 병행하면, 코드 개발과 비즈니스 인프라를 동시에 준비할 수 있다.

**장기**: RGB 통합 컨트롤러가 성공하면, 2순위(3D 프린터 모니터링)와 3순위(범용 계측기 로거)로 서드파티 소프트웨어 포트폴리오를 확장하여, SL Corporation의 "불편한 하드웨어 소프트웨어를 대체하는 회사"라는 브랜드 정체성을 구축할 수 있다.

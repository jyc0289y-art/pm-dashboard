# P10.6.1wc — RGB 통합 컨트롤러 프로젝트: 기술 분석 + 시장 검증

## 메타데이터
- 생성: 2026-03-18
- 포크원: P10.6wc (RGB 통합 컨트롤러 아이템 발굴 세션)
- 자식: 없음

## 작업 요약
P10.6wc에서 포크. RGB 통합 컨트롤러 프로젝트의 기술 분석(OpenRGB 소스코드)과 시장 검증(RGB 문화 인기도 리서치)을 수행. 투자 결정 근거 확보 완료.

## 주요 결과물

### 1. OpenRGB 소스코드 구조 분석 (P10.7wc에 상세)
- **3-파일 패턴**: Controller + RGBController_ + Detect로 140+개 기기 모듈화
- **Corsair HID 프로토콜**: 65바이트 고정 패킷, Command/Property 구조, 3가지 색상 전송 방식 (Direct/Hardware/Software)
- **기기 감지**: Self-registration 패턴 (C++ 정적 초기화), VID/PID 매칭
- **GPL-2.0 회피 전략**:
  - (a) OpenRGB 네트워크 SDK 연동 (포트 6742, MVP용) → GPL 전파 없음
  - (b) 프로토콜 사양만 참조 후 클린룸 구현 (장기) → GPL 전파 없음
  - hidapi는 BSD/GPL/HIDAPI 트리플 라이선스 → BSD 선택 시 안전
- **핵심**: OpenRGB의 "지식"은 참조하되 "코드"는 가져오지 않으면 유료 비즈니스 가능

### 2. RGB 게이밍 문화 인기도 리서치 (P11.1wc에 상세)

#### 시장 데이터

- 프리미엄 제품 중 RGB 탑재 비중: 64%

- Razer Chroma 사용자: 1.5억명+

- Know Your Meme에 "RGB Lights" 공식 등재 (2018~)

#### RGB 소프트웨어 불만 = 우리의 기회
- Tom's Hardware: "Gaming Peripheral Apps, Ranked From Worst to Worst" (전부 최악)

## 총평 — 기술 분석과 시장 검증의 교차 검증으로 투자 결정 근거 확보

### 이번 세션의 의의

P10.6.1wc는 P10.6wc에서 선정한 RGB 통합 컨트롤러 프로젝트의 **투자 결정 근거를 양면에서 확보**한 세션이다. 기술 측면(OpenRGB 소스 분석, P10.7wc)과 시장 측면(RGB 문화 인기도, P11.1wc)을 병렬로 수행하여 "만들 수 있는가"와 "팔 수 있는가"에 동시에 답했다.

가장 중요한 기술적 발견은 **GPL-2.0 회피 전략**의 구체화다. OpenRGB의 "지식"은 참조하되 "코드"는 가져오지 않는 두 가지 경로(네트워크 SDK 연동 또는 클린룸 구현)를 정의하고, hidapi의 트리플 라이선스에서 BSD를 선택하면 유료 비즈니스가 가능하다는 법적 근거를 확보했다. 이는 오픈소스 프로젝트 참조 시 가장 흔하게 간과되는 라이선스 리스크를 선제적으로 관리한 것이다.

시장 데이터도 설득력이 있다. RGB 탑재 비중 64%, 게이머 RGB 선호율 56%, Tom's Hardware의 "전부 최악" 평가, iCUE 4.2GB 디스크 점유 등의 정량적 근거는 "왜 또 다른 RGB 소프트웨어인가"에 대한 명확한 답을 제공한다.

### 전략적 제언

**단기**: Corsair 키보드 구매 + OpenRGB 네트워크 SDK Python 프로토타입으로 MVP 착수 가능.

**중기**: "Peak RGB" 논쟁(일부 제조사가 RGB 없는 옵션 제공)은 모니터링이 필요하지만, LCD 스크린 트렌드로의 진화는 시장 자체의 소멸이 아닌 변형이므로 대응 가능하다.

**장기**: 한국 시장에서 RGB 콘텐츠 소비는 활발하나 소프트웨어 관련 한국어 콘텐츠가 부족하다는 발견은, 한국어 우선 지원을 차별화 전략으로 활용할 수 있음을 시사한다.

## 핵심 결정사항

1. **데모 공유 방식: A+C** — A(영상 업로드→감지→MP4) + C(실시간 CCTV), B(PDF 보고서)는 보조
2. **회사 PC = 서버** — RTX 4060, 24시간 상시 운영, Cloudflare Tunnel로 외부 접속
3. **코드 비노출 원칙** — Streamlit 서버사이드 렌더링, 파트너는 브라우저만 접속
4. **API 키 보안** — `.streamlit/secrets.toml`에만 저장, Git/코드/보고서에 절대 미포함
5. **SUV/밴 오분류 보정 2단계 접근** — 1단계: reclassify_truck() 휴리스틱 (즉시), 2단계: EfficientNet-B0 (Phase 1)
6. **모드C webrtc_streamer 폐기** — 브라우저 웹캠 활성화 버그, cv2.VideoCapture 서버사이드로 전면 교체
7. **회사 PC 작업 위임 체계** — docs/requests/에 의뢰서/지시서 작성 → Git 푸시 → 회사 PC Claude Code 실행

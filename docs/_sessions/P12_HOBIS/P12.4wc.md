## 핵심 결정사항
1. **HVL 값**: Water γ=10.0/n=2.3cm, Paraffin γ=11.0/n=1.9cm
2. **RT룸 수조 핫셀 설계**:
   - 내부: 100×80×100cm (STC-100/УКТIIB-313 수용 가능)
   - 벽/천장: 아크릴 수조(물 30cm) + 납판(5cm)
   - 창: 아크릴 수조(물 30cm) + 납유리(Pb eq. 5cm) — 납판 대신 납유리로 시인성 확보
   - 바닥: SUS304 베이스 + 납판
   - 구조보강: SUS304 50×50×5mm 앵글 프레임 (8주 + 수평 보강재)
3. **핫셀 납유리창 vs 수조창**: 납유리만으로는 중성자 비차폐 (18,092 μSv/h), 물+납유리 조합 필수
4. **54 mCi 시**: 물30cm만으로도 창 기능 가능 (15.86 μSv/h, 납유리 불요)
5. **2.7 Ci 시**: 물30+납유리5cm(14.69) 또는 물40+납유리3cm(24.45) 필요
6. **방사화 평가 조사거리**: 5cm→1cm 변경. 집게로 선원 파지 시 캡슐 중심~집게 표면 최소거리 ≈1cm. 0cm은 1/r² 발산 문제.

## 인터랙티브 차폐 분석 기능 (신규)
- **차폐 단면도 SVG** (`cf252GenerateSchematicSVG`): 선원→차폐층→평가점 횡단면도, 치수선 포함
- **인터랙티브 오버레이** (`cf252OpenOverlay`): 전체화면 모달, 슬라이더로 방사능량/거리/차폐두께 실시간 조절
- **실시간 재계산** (`cf252OvRecalculate`): 슬라이더 변경 시 도면+선량+PASS/FAIL 즉시 갱신
- **차폐층 추가** (`cf252OvAddShieldLayer`): 오버레이 내에서 새 차폐재 추가 가능
- 결과 패널 내 썸네일 SVG 클릭 또는 EXPAND 버튼으로 오버레이 진입
- **CSS 캐시 이슈 해결**: `?v=` 쿼리스트링으로 Python http.server 캐시 우회

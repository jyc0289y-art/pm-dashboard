## 주요 결과물
모든 결과물은 `hobis-v6-dev/` 디렉토리 기준:
- `js/cf252.js` — Cf-252 계산 엔진 (감마/중성자 선량률, 다중 선원, 방사화 평가, 보고서 기반 프리셋 8종)
- `js/cf252-ui.js` — Cf-252 UI 모듈 (모드 전환 DOM 보존, 프리셋 로드 + reportRef 표시, 결과 렌더링, getState/restoreState)

- `js/log.js` — 영구 로그 (IndexedDB, GPS 좌표, ISO 일시)
- `js/core.js` — 탭 상태 저장/복원, Cf-252 DOM 보존 (cf252Initialized 플래그)
- `js/ri-order.js` — CSV/XLS/HTM/frameset 다중 포맷 파싱 + XSS 이스케이프 수정
- `index.html` — Cf-252 탭 + 패널, CSP 메타 태그 추가
- `css/hobis.css` — Cf-252 스타일 + GPS 로그 + 프리셋 참조 스타일

## 핵심 결정사항
1. 기존 calculator.js(범용)와 별도의 독립 모듈(cf252.js + cf252-ui.js)로 구현
2. 감마상수: Smith & Stabin 2012 기반 22.176 mSv·cm²/(mCi·h) — GAMMA_CONST_MSV로 getter 유도
3. 중성자 환산계수: KAERI 1993 H/Φ=332.9 pSv·cm²/n 기반으로 자동 유도
4. HVL은 6차 보완 최종 채택값 적용 (혼합방사선장 보수적 값)
5. STC-100 용기 차폐(PE 24.65cm)는 CONTAINERS 객체에서 자동 적용
6. 방사화 평가: Al 체적(기본 10cm³)을 사용자 입력 가능하도록 구현
7. Al-27 (n,γ) 단면적: 1 mb (보수적, ENDF/B 기준)
8. 탭간 DOM 보존: cf252RenderInputs()에서 dose/activation 양쪽 wrap을 동시 생성, show/hide 토글
9. 상태 유지: cf252GetState() → IDB 저장 → 리로드 시 cf252RestoreState()로 입력값+결과 HTML 복원
10. 작업 디렉토리 분리: `_hobis-v6-ref/`(원본) ↔ `hobis-v6-dev/`(개발)
11. **안전보고서(HJSR-01 REV.13) 기반 프리셋 보정**:
    - 핫셀 거리: 200cm → 224cm (√((1.325²+1.8²))=2.24m)
    - 핫셀 콘크리트: 60cm(천장값) → 120cm(벽)/135cm(C1-DP2)/60cm(천장) 3종 분리
    - 저장시설: 콘크리트 벽 120cm extraShielding 추가
    - 프리셋 5종 → 8종 확장 (퍼 표면/벽외부, 허 표면/벽외부, 핫셀 벽/C1-DP2/천장, 방사화)
    - 각 프리셋에 reportRef 필드로 보고서 페이지 및 기준선량률 명시
12. **보안성검토 수행 및 수정**:
    - High: ri-order.js innerHTML XSS → fcEsc() 적용 완료
    - Medium: CSP 부재 → meta CSP 추가 완료
    - Medium: CDN SRI 없음 → 향후 로컬 번들링 권장
    - Medium: GPS 프라이버시 → 토글 UI 권장 (미구현)

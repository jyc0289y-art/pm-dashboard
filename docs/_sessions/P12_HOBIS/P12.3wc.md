## 핵심 결정사항
1. **감마상수**: 보고서 1R=10mSv 단순환산 → F_FACTOR(0.960) 미적용이 정확
2. **Al-27 단면적**: ENDF/B 0.512mb (보고서 p.97 일치)
3. **핫셀 C1-DP1,3,4**: 콘크리트 경로 1,615mm (도면 벽두께 1,200mm와 다름 — p.95 계산식이 권위)
4. **차폐역산**: γ+n 복합방사선장 → bisection 수치해 (서로 다른 HVL 때문에 해석해 없음)
5. **감쇄곡선**: 로그 스케일 Y축, γ/n/합계/기준선 4개 데이터셋
6. **PDF**: window.open() + print() 방식 (팝업 차단 시 HTML 다운로드 폴백)

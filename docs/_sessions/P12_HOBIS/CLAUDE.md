# HOBIS Cf-252 프로젝트지침

## 프로젝트 개요
- **목적**: Cf-252 중성자원 기반 차폐/선량평가 계산기 (웹 앱)
- **P번호**: P12 (P12.{N}wc)
- **최신 세션**: P12.11wc (2026-04-10, PM 파견 탐색)
- **실질 최신 작업**: P12.10wc (시각화 범용화 SHIELD/DECAY 탭)
- **배포**: GitHub Pages (jyc0289y-art)

## 기술 스택
- HTML/CSS/JS (순수, 빌드 도구 없음)
- Three.js (3D 뷰어: 수조/콘크리트 핫셀)
- SVG (단면도, 인터랙티브 오버레이)
- Python (보조 계산 스크립트)

## 핵심 기능
- **Cf-252 선량평가**: 프리셋 27종 기반 실시간 계산 (조사거리 1cm 보정 포함)
- **3D 시각화**: 수조/콘크리트 핫셀 Three.js 뷰어, 내부용적, 사람모델, 치수선
- **SVG 단면도**: 인터랙티브 오버레이, 실시간 선량 계산 연동
- **HVL 교차검증**: NIST XCOM 기반 Pb/Al/Fe/PE/파라핀 검증 완료

## 세션 관리
- registry: `memory/sessions/registry.md`
- 포크 시 `P12.{N}wc` 형식

- P12.1~5wc: Cf-252 전용 기능 완성 (프리셋, 3D, SVG, 선량평가)

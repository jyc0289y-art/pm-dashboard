### 3. SHIELD/DECAY 시각화 범용화 진행 상태

**결론: calc-viz.js에서 범용화가 이미 상당히 진행되어 있으며, 핵심 기능은 동작 가능 상태.**

#### 완료된 기능 (calc-viz.js, 811줄)
1. **SHIELD SVG 단면도** (`shieldGenerateSVG`, L14-150): 선원/차폐층/평가점을 스케마틱으로 SVG 렌더링. 11종 재질 컬러 매핑 완비 (Pb, Concrete, Steel, W, DU, LeadGlass, Water, PE, Paraffin, Al, Iron)
2. **SHIELD 인터랙티브 오버레이** (`shieldOpenOverlay`, L157-365): 방사능/거리/차폐층 두께를 슬라이더+숫자입력으로 실시간 조절. SVG 실시간 업데이트 + 결과 카드. Cf-252 오버레이 CSS 클래스(`cf252-overlay`, `cf252-ov-*`) 재사용.
3. **DECAY 인터랙티브 오버레이** (`decayOpenOverlay`, L372-615): 초기 방사능/경과일 슬라이더. Chart.js 감쇠 곡선 + 결과점 마킹. 실시간 재계산.
4. **레퍼런스 테이블** (`generateFullRefHTML`, L620-711): 감마상수 비교, HVL 비교, 타당성 검증, 참조문헌 4개 섹션
5. **PDF 내보내기** (`calcExportPDF`, L718-811): DECAY/SHIELD 결과를 인쇄용 HTML로 출력. SVG + 차트 이미지 포함.

#### 통합 상태
- calculator.js에서 `decayOpenOverlay()`, `shieldOpenOverlay()` 호출부 확인 (L56-57, L149-151)
- SHIELD forward 결과에 SVG 썸네일 + "EXPAND" 버튼 → 오버레이
- DECAY forward 결과에 "INTERACTIVE VIEW" 버튼 → 오버레이
- index.html에 PDF 버튼 배치 확인 (L305)

#### 미완/미확인 사항
- P12.10wc 세션지침이 "진행중"으로 남아 있고 결과물이 비어 있어 추가 작업 의도 불명
- P12.6wc (범용화 시작 세션) 세션지침 부재 — 원래 목표 범위 대비 완성도 비교 불가
- SHIELD reverse, DECAY reverse에 대한 오버레이/시각화는 미구현 (forward만 지원)
- 3D 뷰어(hotcell-3d.html, hotcell-concrete-3d.html)는 Cf-252 전용으로 범용화 대상 외

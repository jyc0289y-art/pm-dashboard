# P12.8wc — 파라핀 중성자 HVL 문헌조사

## 메타데이터
- 생성: 2026-03-26
- 포크원: P12.7wc
- 자식: 없음

## 작업 요약
파라핀 왁스(C25H52, ρ=0.93)의 Cf-252 중성자/감마 차폐 데이터를 문헌 검색으로 수집.
직접적인 파라핀 Cf-252 HVL 논문은 미발견이나, PE와의 수소밀도 유사성(~2.6% 차이)으로
PE HVL(1.85cm, Rahvar 2020)을 파라핀에 동일 적용 가능함을 근거와 함께 정리.

## 주요 결과물
- 파라핀 neutron HVL: ~1.85 cm (PE 기반 추정, Rahvar 2020)
- 파라핀 ΣR: 0.123 cm-1, ΣR/ρ: 0.132 cm2/g (원소별 혼합법칙 계산)
- 파라핀 gamma HVL: 9.2 cm (0.8 MeV), 10.3 cm (1.0 MeV) — NIST XCOM
- PE vs 파라핀 비교표, 문헌 목록 정리

## 핵심 결정사항
- 파라핀 neutron HVL = PE HVL = 1.85 cm 사용 가능 (수소밀도 차이 <3%)
- removal cross section HVL(~5.6cm)와 dose HVL(1.85cm)의 차이는 물리적으로 다른 양임을 명확히 구분

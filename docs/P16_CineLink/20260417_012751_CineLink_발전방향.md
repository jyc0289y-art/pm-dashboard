# CineLink (P16) 영상 파이프라인 통합 발전방향

**작성일**: 2026-04-17 01:27:51
**세션**: P16.3wc
**승인**: 사용자 확정 (Plan Mode 종료)

> **핵심 취지**: **옵션 1 → 옵션 2 → 옵션 3 진화 로드맵** + TEREV 첫 파일럿
>
> 옵션 1(CapCut 전담)은 **도착지가 아니라 출발점**이다. 단기 안정성을 확보하면서 옵션 2(이원 경로) → 옵션 3(CineLink 전담 완전 자동화)을 향해 점진적으로 이행한다. `ffmpeg_lib.py`의 `build_composite()` 등 연구라인은 "언젠가 쓸 수도 있는 사이드 프로젝트"가 아니라 **최종 목적지로 가는 레일**이다.

---

## Context — 왜 이 변경이 필요한가

CineLink(P16)는 P16.1wc에서 파이프라인 설계를 완료했으나 실전 영상 0건 상태로 정체 중이다. 이번 세션(P16.3wc)에서 두 가지 입력이 추가 확보되었다:

1. **`capcut-claude-hybrid-strategy.md`** — 빌더 조쉬×노정호의 3층 아키텍처 (Claude Code 두뇌 → CineLink 전처리 → CapCut 조립). "편집자 없이 굴러가는 시스템" 비전.
2. **FEP(P21)의 FFmpeg TRAP 11개** — 영상 합성 단계에서 반복 발생한 함정의 체계화된 카탈로그. 동일 실수 반복 방지 가능.

현재 CineLink는:
- `cinelink/` 스캐폴딩 존재, 5종 Agent 정의 완료
- Whisper ❌, LUT ❌ 등 블로커 미해소
- TRAP 시스템·FFmpeg 표준 라이브러리·Skills/Commands 체계 부재

본 계획은 하이브리드 전략 **부분 수용**(CapCut 전담 + CineLink 연구라인 유지) + **FEP TRAP 11개 즉시 내재화** + **TEREV 첫 파일럿** 3축으로 CineLink를 "설계 단계"에서 "실전 투입 단계"로 전환한다.

## 거부/보류 사항

- **TTS: Typecast(ko) + ElevenLabs(ja) 조합** — 거지맵은 현장 오디오 기반이라 TTS 불필요. TEREV만 Typecast 채택, ElevenLabs 보류.
- **`~/CineLink/{channel}/` 물리 경로 평행 이동** — iCloud 경로와 충돌, 현행 유지.
- **CapCut API 연동 가정** — 공식 CLI 없음, 초기 설계 가정 배제.
- **A/B 테스트·댓글 분석 자동화** — 실전 영상 3~5편 누적 후 재검토.

## CL-TRAP 카탈로그 (11개, FEP→CineLink 이관)

| CL-TRAP | 원본 FEP | 제목 | 우선순위 | 대응 함수 |
|---------|---------|------|---------|----------|
| 001 | TRAP-011 | `-shortest` 금지, `-t {duration}` 명시 | Critical | `trim_exact()` |
| 002 | TRAP-012 | concat glob에서 `_temp_`, `_seg`, `_cut` 제외 | Critical | `concat_clips()` |
| 003 | TRAP-013 | concat 오디오 파라미터 일치 (`-ar 24000 -ac 1 -c:a aac`) | Critical | `normalize_audio_for_concat()` |
| 004 | TRAP-020 | Chromakey 확정값 `0x30C010:0.12:0.02` | Critical | `chromakey_lp_green()` |
| 005 | TRAP-034 | 멀티세그 빌드 오버레이 누락 방지 | Critical | `build_composite(mode="unified")` |
| 006 | TRAP-014 | drawtext 한글 미지원 → Pillow | High | `draw_text_cjk()` |
| 007 | TRAP-015 | zoompan은 비디오 입력 불가 → scale+crop | High | `ken_burns_from_image()` |
| 008 | TRAP-032 | overlay `shortest=1` 금지 → `eof_action=repeat` | High | `overlay_with_repeat()` |
| 009 | TRAP-033 | concat + 외부 오디오 `-map` 명시 필수 | High | `mux_concat_with_external_audio()` |
| 010 | TRAP-038 | manifest → `silencedetect` auto-snap | High | `snap_manifest_to_silence()` |
| 011 | TRAP-039 | 말풍선 꼬리 Pillow 커스텀 렌더링 | High | `render_speech_bubble()` |

**게이트 전략**: `ffmpeg-runner.md`에 "FFmpeg 명령 생성 전 `shared/scripts/ffmpeg_lib.py` 확인, 있으면 함수 호출. 키워드(concat/chromakey/drawtext/overlay/zoompan) 감지 시 `docs/production_lessons.md` CL-TRAP-ID 회피책 적용" 2줄 추가.

## 실행 로드맵 (Phase A → B → C → D, 옵션 1→2→3 진화)

```

   │
   ▼
```

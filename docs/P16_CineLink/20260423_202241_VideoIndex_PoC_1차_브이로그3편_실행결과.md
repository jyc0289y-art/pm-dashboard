## 파이프라인 구성

1. **faster-whisper base 모델** — 언어 자동 감지 + 단어별 타임스탬프
2. **PySceneDetect ContentDetector (threshold=27)** — 화면 전환
3. **FFmpeg silencedetect** (`noise=-30dB, d=0.5s`) — 무음 구간
4. **가중 결합** (scene 0.5 + silence 0.3, threshold 0.35)
5. **FFmpeg re-encode 분할** — CRF 20, +faststart
6. **MD 렌더** — INDEX.md + 클립별 MD + transcript.json

## 📋 검토 체크리스트

영상 + MD를 보시면서 확인해볼 포인트:

### 영상(.mp4) 검토
- [ ] 클립 시작·종료 지점이 **자연스러운 컷**인지 (중간 잘림 vs 문장 완성)
- [ ] duration과 내용이 **의도한 편집 단위**인지 (너무 짧음/길음)
- [ ] 같은 장면 연속 클립이 **반복되는지** 또는 **다른 맥락인지**

### MD 파일 검토
- [ ] **클립별 MD의 "대사"**가 실제 영상 대사와 일치하는지
- [ ] **"감지된 객체"** 목록이 실제 화면의 주요 오브젝트와 맞는지
- [ ] INDEX.md의 **클립 목록 표**가 훑어보기로 편집 계획 수립에 유용한지

### 파일명 검토
- [ ] 파일명만 보고 **어떤 장면인지 추측 가능**한지
- [ ] **불용어 필터**가 잘 작동하는지 (의미 있는 단어 중심)
- [ ] **객체 태그**가 장면 이미지 떠올리는 데 도움이 되는지

### 전체 관점
- [ ] **찾고 싶은 특정 장면**을 INDEX.md에서 **몇 초 안에 찾을 수 있는지**
- [ ] 대사 없는 구간(fallback `clipNN`)을 **허용 가능한지** (11/169 = 6.5%)
- [ ] 재실행 시 **파일명이 바뀌는 문제** (촬영 시각 접두사가 재실행 때마다 갱신)

## 💬 검토 후 피드백 요청

영상·MD를 직접 보신 후 알려주시면 좋을 사항:

1. **파일명이 실제 편집에 도움이 되는가?** (이 표기법 유지 or 변경)
2. **클립 길이 단위(평균 12초)가 적절한가?** (더 길게/짧게)
3. **대사 오전사가 실전 방해 수준인가?** (Whisper large-v3 교체 필요?)
4. **객체 태그가 유용한가?** (장소 태그 추가 시 가치 확인 = Phase B-5)
5. **없는 기능 중 필요한 것?** (예: 화자 구분, 썸네일, 하이라이트 자동 추출)

**작업 목록:**
1. `pyproject.toml` 작성 (의존성: fastapi, uvicorn, sqlalchemy, pydantic, httpx, apscheduler, cachetools, python-telegram-bot)
2. `.env.example` 작성 (TEQUILA_API_KEY, AMADEUS_KEY/SECRET, TELEGRAM_BOT_TOKEN 등)
3. `docker-compose.yml` (backend + nginx, Phase 3에서 redis 추가)
4. `.gitignore` 작성

### Step 2. DB + 모델

1. `database.py` — SQLAlchemy async engine + sessionmaker (aiosqlite)
2. 모델 6개 정의 (User, WatchedRoute, PriceRecord, TrackedFlight, AlertConfig, ApiUsage)
3. Alembic 마이그레이션 셋업 (추후 PostgreSQL 전환 대비)
4. 초기 마이그레이션 생성 + 적용

### Step 3. Tequila API 클라이언트

1. `tequila_client.py` — httpx AsyncClient 기반
   - `search_flights(origin, dest, date_from, date_to, return_from?, return_to?)` → 검색 결과
   - `search_explore(origin, budget, date_from?, date_to?)` → "어디든" 탐색
2. 응답 파싱 → PriceRecord 스키마 변환
3. 캐시 래퍼: TTLCache(maxsize=500, ttl=1800) — 동일 검색 30분 캐시
4. API 사용량 로깅 (ApiUsage 테이블 기록)

### Step 4. 검색 API

1. `GET /api/search/flights` — Tequila 검색 → 캐시 확인 → 결과 반환
2. `GET /api/search/explore` — Tequila "어디든" → 목적지별 최저가 리스트
3. 응답 스키마: 항공사, 가격(원본+KRW), 출발/도착 시간, 경유 수, 소요시간, 예매 링크

### Step 5. 모니터링 CRUD + 스케줄러

1. `POST /api/watch` — 노선 등록 (user_id는 device_id에서 자동 매핑)
2. `GET /api/watch` — 등록된 노선 목록 (최근 최저가 포함)
3. `DELETE /api/watch/{id}` — 모니터링 해제
4. `GET /api/watch/{id}/history` — 가격 히스토리 (페이지네이션)
5. APScheduler: 등록된 노선별 check_interval마다 Tequila 호출 → PriceRecord 저장
6. 가격이 max_price 이하이면 알림 큐에 추가

### Step 6. API 사용량 추적

1. 미들웨어 또는 데코레이터로 모든 외부 API 호출 시 ApiUsage 기록
2. `GET /api/system/api-usage` — API별 사용량 요약 (일/월 단위)
3. `GET /api/system/health` — DB 연결 + 각 API 상태 체크

## Phase 1b: 프론트엔드 + 알림 (목표: 1주)

### Step 7. Next.js 셋업

1. `create-next-app` (App Router, TypeScript, Tailwind)
2. 공통 레이아웃: 헤더(로고 + 네비게이션) + 모바일 반응형
3. API 클라이언트 유틸 (fetch wrapper, 에러 핸들링)
4. device_id 관리 (localStorage UUID, 첫 방문 시 자동 생성 → 백엔드 User 생성)

### Step 8. 검색 페이지

1. **검색 폼**: 출발지(IATA 자동완성) + 도착지 + 날짜 범위 + 편도/왕복
2. **결과 리스트**: 카드 형태 (항공사 로고, 가격, 시간, 경유 정보)
3. **정렬/필터**: 가격순, 시간순, 직항만, 항공사 필터
4. **예매 버튼**: Tequila Affiliate 링크 (새 탭)

### Step 9. "어디든" 탐색 페이지

1. **입력**: 출발지 + 예산 (슬라이더 또는 입력)
2. **결과**: 목적지 카드 그리드 (도시 사진 + 최저가 + 날짜)
3. 카드 클릭 → 해당 노선 상세 검색으로 이동

### Step 10. 모니터링 대시보드

1. **노선 목록**: 등록된 모니터링 노선 카드
2. **각 카드**: 노선명, 현재 최저가, 목표가, 마지막 체크 시간
3. **가격 차트**: Recharts 라인 차트 (일별/주별 추세)
4. **노선 등록 모달**: 검색 결과에서 "모니터링 추가" → 목표가/주기 설정

### Step 11. 텔레그램 알림

1. `telegram_bot.py` — python-telegram-bot 또는 직접 HTTP
2. 알림 설정 페이지: 텔레그램 chat_id 입력 (봇 시작 → /start → chat_id 안내)
3. 알림 메시지 포맷:
   ```
   ✈️ 가격 알림!
   ICN → NRT (2026-05-01)
   현재 최저가: ₩185,000 (목표: ₩200,000)
   항공사: 피치항공 (직항)
   👉 예매하기: [link]
   ```
4. 알림 히스토리 페이지 (`GET /api/alerts/history`)
